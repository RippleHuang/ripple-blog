## 项目描述

1. 项目为 网易云 Web App (SPA)
2. 主要技术栈
    - 前台数据处理交互
        - vue
        - vue-router
        - vuex
        - vue-lazyload
        - vant-ui
    - 前后台交互
        - axios
        - API接口:NeteaseCloudMusicApi
    - 模块化
        - ES6
        - babel
    - 项目构建/工程化
        - webpack
        - vue-cli
        - eslint
    - css预处理理器
        - sass
